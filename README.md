# 📦 InventoryHub

A warehouse and inventory management system developed as a .NET console application using **C#** and **Visual Studio**.

---

## 🚀 Overview

InventoryHub is designed to manage warehouse operations in a simple and structured way. It demonstrates the use of **Object-Oriented Programming (OOP)** along with modern tools like **MongoDB** and **Docker**.

---

## 🧩 Features

### 👨‍💼 Admin Panel

* Manage products/Inventory (Add, Update, Delete, View)
* Manage suppliers
* Manage Orders
* View reports (Low stock, revenue chart, etc.)
* Database Stats

### 🛒 Customer Panel

* Browse and search products
* Add items to shopping cart
* Place orders
* View order history

---

## 🛠️ Technologies Used

* 💻 C# (.NET Console Application)
* 🧠 Object-Oriented Programming (OOP)
* 🗄️ MongoDB (NoSQL Database)
* 🐳 Docker (Containerization)
* 🧰 Visual Studio

---

## 🎯 Key Highlights

* Clean and modular code structure
* Real-world inventory management workflow
* Separate Admin and Customer modules
* Database integration with MongoDB
* Docker-based deployment support

---

## ▶️ How to Run

In DatabaseHelper.cs in line 20,21 add mongodb string and database name.You can find the database name in appsettings.json file then write your own mongodb string and run the project.
